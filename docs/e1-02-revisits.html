Currently every sample that landed in a tree is present in the data.

For this fr, the dv is true if the tree has been visited before in the trial, and false if it’s the first time they have visited the tree.

Having obtained fr number of fruit, and while searching for the fr+1th fruit, how many times did they revisit a tree that they had previously visited in the trial?

The question expressed this way allows us to see that while fr == 0 they are looking for the first fruit, and so revisits are possible; whereas as soon as fr becomes 10 the trial is over: it’s not possible to do any revisits while looking for the 11th fruit. The dv is always false for fr of 10. While fr==0 it’s possible to get NA revisits: this is the case when they get a fruit on the first tree they look at.

We remove samples that landed in the 10th fruit.

e1.revisits <- e1 %>% 
  filter(fr != 10)

14,291 rows

At this point the data have a row for each sample that landed in a tree except the final sample where fr becomes 10.

The next step shrinks the data by switching from true/false for whether this sample was a revisit, to a count of how many revisits there were while searching for the next fruit. Only some trials have counts while fr is 0 – the count can be zero if they don’t revisit any trees on the way to finding the first fruit if they visit some non-fruit-bearing trees on the way. The highest count for fr of 0 is 4 – they revisited 4 trees that didn’t bear fruit before finding the first fruit-bearing tree. If they didn’t fixate any non-fruit-bearing trees before fixating the first fruit, that row is simply a structural missing. Hence this df has 7783 rows whereas if there were no structural missing there would be 42210*10=8400 (after bringing fr down to 0:9).

e1.revisit.counts.sparse <- e1.revisits %>% 
  group_by(across(c(ex, pp, rr, st, tb, fr))) %>% 
  summarise(n_revisits_per_fruit = sum(is_a_revisit)) %>% 
  ungroup()

7,783 rows

An issue here is that the resulting matrix is sparse unless we join it to a matrix representing the experimental design, so we have to do the full_join

  • Matrix representing the full design
e1.design = tibble(
  expand_grid(ex = 1, 
              pp = unique(e1.revisits$pp), 
              rr = sort(unique(e1.revisits$rr)), 
              tb = unique(e1.revisits$tb),
              st = "", 
              fr = 0:9) %>% 
    mutate(st = ifelse(tb <=5, "early", "late"))
)
  • full_join to make the matrix non-sparse
e1.revisit.counts.notsparse <- 
  full_join(e1.design, e1.revisit.counts.sparse) %>% 
  arrange(pp, rr, tb, fr)

Now we have 8400 rows. The padding inserted NA for fr of 0 for trials where there were no revisits on the way to the first fruit. This happened 409 times.

We say that if revisits is NA then we convert the NA to zero. Thus embodies the opinion that they did not in fact do any revisits, even though getting a fruit on the first tree ‘denied’ them the possibility of racking up a revisit while fr was 0.

e1.revisit.counts.notsparse <- e1.revisit.counts.notsparse %>% 
  replace_na(list(n_revisits_per_fruit = 0))

We have 8400 cases made by fully crossing these: * pp (42) * rr (2) * tb (10) which is st (2) * tb[1:5] or tb[6:10] * fr (10)

First we take the average, over each of 5 trials, of the count at each fruit-gathering episode.

kv = e1.revisit.counts.notsparse %>% 
  group_by(pp, rr, st, fr) %>% 
  summarise(nr=mean(n_revisits_per_fruit))

Factor it.

kv <- kv%>% mutate(pp=as_factor(pp), rr=as_factor(rr), st=as_factor(st), fr=as_factor(fr))

This is now 42 * 2 * 2 * 10 and suitable for ANOVA
